# java-renter-car
###기능구현
- 차량은 소나타 2대, 아반떼 1대, k5 2대
```
각 차량의 연비
* Sonata : 10km/리터
* Avante : 15km/리터
* K5 : 13km/리터

```
- 이동거리 입력
- 차량 별로 주입해야 할 연료량
