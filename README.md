# java-renter-car
###기능구현
- 차량은 소나타 2대, 아반떼 1대, k5 2대
```
각 차량의 연비
* Sonata : 10km/리터
* Avante : 15km/리터
* K5 : 13km/리터

```
- 이동거리 입력
- 차량 별로 주입해야 할 연료량

# java-coordinate
- 좌표 입력
- split (,)(-)
- x,y (~24 까지)
- 입력 범위 초과시 다시 입력
- 정상적인 입력시 해당 좌표에 특수문자 입력
- 좌표값에 따른 선,삼각형,사각형 나누기
- 직선인 경우는 거리
- 도형인 경우는 면적
